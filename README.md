# Campus Lost & Found

Android mobile application to help students and staff report and recover lost items on campus.

## Tech stack
- Java
- Android Studio
- Realm DB
- XML layouts
- RecyclerView
- Fragments
- GitHub

## Package structure
- model/
- db/
- utils/
- adapter/
- fragment/
